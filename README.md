### checkIfFileExists

```
checkIfFileExists(AppConfig.REPO_PATH+path+fileName+".txt");
 
private static boolean checkIfFileExists(String filePath){
	File f=newFile(filePath);
	if(f.exists()&&!f.isDirectory()){
		return true;
	}
	return false;
}
```
